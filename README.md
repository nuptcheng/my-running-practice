## 记录的数据指标

每次跑步记录包含以下信息：
- **日期时间**：跑步的日期和时间
- **距离**：跑步距离（公里）
- **时长**：跑步时长（分钟）
- **配速**：平均配速（分/公里）
- **心率**：平均心率（bpm）
- **最大心率**：最大心率（bpm）
- **体重**：当天体重（kg）
- **天气**：天气情况
- **感受**：主观感受评分（1-10）
- **备注**：其他备注信息

## 快速开始

### 1. 查看训练计划

打开 [本月训练计划](training-plans/2026-01-plan.md)，了解本周的训练安排。

### 2. 按计划训练

根据计划进行训练，注意控制心率和配速。

### 3. 记录跑步数据

**方法一：快速记录工具（推荐）**

```bash
python scripts/quick_log.py
```

工具会自动：
- 引导输入数据
- 计算配速
- 对比训练计划
- 生成记录行

**方法二：手动记录**

在 `data/2026/01-January.md` 中添加记录。

### 4. 查看数据分析

运行分析脚本生成统计报告和图表：

```bash
# 安装依赖
pip3 install pandas matplotlib seaborn

# 运行分析脚本
python3 scripts/analyze.py

# 生成可视化图表
python3 scripts/visualize.py

# 体重-心率-配速关联分析
python3 scripts/weight_analysis.py
```

### 5. 查看进步情况

分析脚本会生成：
- 月度跑量统计
- 配速变化趋势
- 心率区间分布
- 体重变化曲线
- 训练强度分析
