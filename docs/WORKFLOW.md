## 日常使用流程

### 每次训练（5分钟）

1. **训练前**（1分钟）
   ```bash
   # 打开训练计划，查看今日安排
   open training-plans/2026-01-plan.md
   ```

2. **训练中**（30-60分钟）
   - 按计划执行
   - 监控心率
   - 控制配速

3. **训练后**（3分钟）
   ```bash
   # 快速记录
   python scripts/quick_log.py

   # 输入数据后，复制生成的记录行
   # 粘贴到 data/2026/01-January.md
   ```

4. **更新计划**（1分钟）
   - 在训练计划中标记 ✅ 已完成
   - 填写实际完成情况

### 每周回顾（10分钟）

1. 查看本周完成情况
2. 对比计划目标
3. 调整下周计划（如需要）

### 每月总结（30分钟）

1. 运行数据分析
   ```bash
   python scripts/analyze.py
   python scripts/visualize.py
   ```

2. 查看生成的图表（output/目录）

3. 填写月度总结
   - 完成情况
   - 收获与进步
   - 需要改进
   - 下月计划

## 关键提醒

### ⚠️ 训练原则
1. **心率优先**：心率控制比配速更重要
2. **循序渐进**：每周跑量增加不超过10%
3. **及时记录**：训练后立即记录，避免遗忘
4. **定期回顾**：每周查看进度，及时调整

### 💡 效率技巧
1. 使用快速记录工具，节省时间
2. 训练前截图计划，训练时参考
3. 设置手表/手环心率提醒
4. 定期备份数据到云端

### 📈 进步追踪
- 每周对比心率变化
- 每月查看配速趋势
- 关注主观感受改善
- 记录体重变化

## 常见场景处理

### 场景1：心率超标怎么办？
1. 立即降低配速
2. 记录实际数据
3. 下次训练放慢速度
4. 在训练计划中标注改进点

### 场景2：没有完成计划怎么办？
1. 如实记录实际完成情况
2. 分析原因（身体、天气、时间等）
3. 调整后续计划
4. 不要强行补偿，避免过度训练

### 场景3：感觉太轻松怎么办？
1. 保持当前配速
2. 观察心率是否稳定
3. 连续3次轻松后再考虑提速
4. 优先确保心率达标

### 场景4：跑步机和户外如何选择？
- **跑步机优势**：
  - 配速稳定，易于控制
  - 不受天气影响
  - 适合心率训练

- **户外优势**：
  - 更接近实战
  - 风景变化，不枯燥
  - 地形变化，训练更全面

建议：初期以跑步机为主，掌握心率控制后增加户外训练。
