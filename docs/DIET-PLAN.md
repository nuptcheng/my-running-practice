# 跑步减重饮食计划

> **重要提示**：本饮食计划仅供参考，不能替代专业营养师的建议。如有健康问题，请咨询医生或营养师。

## 个人信息

- **年龄**：42岁
- **当前体重**：81kg
- **目标体重**：75-78kg
- **减重目标**：3-6kg
- **时间规划**：6-8周
- **训练强度**：每周3-4次跑步

## 饮食原则

1. **热量控制**：每天减少300-500卡（不要过度）
2. **蛋白质优先**：每kg体重1.5-2g（120-160g/天）
3. **碳水控制**：一天一顿主食（午餐）
4. **多喝水**：每天2-3升
5. **少油少盐**：避免重口味

## 外食技巧

### 快餐店选择

**推荐**：
- 赛百味（选全麦面包，多蔬菜）
- 沙拉店
- 日式料理（刺身、烤鱼）
- 轻食餐厅

**避免**：
- 麦当劳、肯德基（油炸）
- 中式快餐（太油）

### 点餐技巧

1. **要求少油少盐**
2. **多要蔬菜**
3. **选择清蒸、水煮、烤的烹饪方式**
4. **避免勾芡、油炸**
5. **米饭要小碗**

## 减重时间线

### 第1-2周

- 体重可能下降1-2kg（主要是水分）
- 适应新的饮食习惯
- 可能会感到饥饿，正常现象

### 第3-4周

- 体重稳定下降（每周0.5-1kg）
- 身体适应，饥饿感减少
- 开始看到体型变化

### 第5-8周

- 持续减重
- 体重降至78-79kg
- 跑步心率开始下降

## 配合训练的饮食调整

### 训练日

- 早餐可以多吃一点
- 跑步前1-2小时吃香蕉
- 跑步后补充蛋白质
- 晚餐可以多吃蔬菜

### 休息日

- 热量可以稍微减少
- 保持蛋白质摄入
- 多喝水

## 常见问题

### Q1: 晚上饿了怎么办？

A: 可以吃：
- 黄瓜
- 西红柿
- 水煮蛋
- 无糖酸奶
- 不要吃：米饭、面包、水果

### Q2: 周末聚餐怎么办？

A:
- 选择烤肉、火锅（少油）
- 多吃肉和蔬菜
- 少吃主食
- 不喝酒或少喝
- 第二天恢复正常饮食

### Q3: 感觉没力气跑步？

A:
- 可能热量太低，适当增加
- 跑步前吃香蕉
- 午餐可以多吃一点

### Q4: 体重不下降怎么办？

A:
- 检查是否严格执行
- 可能是平台期，继续坚持
- 增加运动量
- 减少隐藏热量（调料、零食）

## 重要提醒

1. **不要过度节食**：每天至少1800卡
2. **保证蛋白质**：肌肉很重要
3. **多喝水**：每天2-3升
4. **循序渐进**：不要急于求成
5. **听从身体**：如果太累，适当增加热量
6. **咨询专业人士**：如有疑问，咨询营养师
