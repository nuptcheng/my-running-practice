# 快速开始示例

本文档演示如何使用这个跑步训练系统。

## 场景：今天完成了一次跑步训练

### 步骤1：查看今日计划

打开 `README.md` 或 `training-plans/2026-01-plan.md`，查看今天的训练计划。

例如：
```
周三 1/15
- 计划: 轻松跑 5km，心率<138，配速7:30-8:00
```

### 步骤2：完成训练

按照计划进行训练，注意：
- 实时监控心率
- 控制配速
- 记录感受

### 步骤3：快速记录

训练完成后，运行快速记录工具：

```bash
python3 scripts/quick_log.py
```

按提示输入数据：

```
============================================================
跑步记录录入
============================================================

日期 (直接回车使用今天): [直接回车]
距离 (km): 5.2
时长 (分钟): 42
计算配速: 8:04
平均心率 (bpm): 135
最大心率 (bpm): 148
体重 (kg): 80.5

场地类型:
1. 跑步机
2. 户外
3. 操场
4. 其他
选择场地 (1-4): 2

感受评分 (1-10): 8
备注: 状态不错，心率控制得很好
```

### 步骤4：查看对比结果

工具会自动显示与计划的对比：

```
============================================================
与训练计划对比
============================================================

📊 本次训练数据:
  距离: 5.2 km
  配速: 8:04
  心率: 135 bpm

🎯 训练计划目标:
  心率: < 138 bpm
  配速: 7:30 - 8:00

✅ 完成情况:
  ✅ 心率达标: 135 bpm
  ⚠️  配速偏慢: 8:04
     说明: 配速慢没关系，心率控制更重要

============================================================
```

### 步骤5：更新记录文件

复制生成的记录行：

```
| 2026-01-15 | 5.2 | 42.00 | 8:04 | 135 | 148 | 80.5 | 户外 | 8 | 状态不错，心率控制得很好 |
```

粘贴到 `data/2026/01-January.md` 的表格中。

### 步骤6：更新训练计划

在 `training-plans/2026-01-plan.md` 中更新：

```markdown
### 周三 1/15 ✅ 已完成
- **计划**: 轻松跑 5km，心率<138，配速7:30-8:00
- **实际**: 户外 5.2km，42分钟，配速8:04，心率135
- **场地**: 户外
- **对比分析**:
  - ✅ 心率达标（135 vs 138目标）
  - ✅ 距离达标
  - ⚠️ 配速稍慢但可接受
- **改进建议**: 继续保持，心率控制很好
```

### 步骤7：查看进度

打开 `README.md`，查看本周和本月的完成进度。

## 每周/每月分析

### 每周日晚上

回顾本周训练：
1. 打开训练计划，查看本周完成情况
2. 统计本周跑量、平均心率
3. 规划下周训练

### 每月最后一天

运行数据分析：

```bash
# 生成统计报告
python3 scripts/analyze.py

# 生成可视化图表
python3 scripts/visualize.py
```

查看生成的图表（在 `output/` 目录）：
- `distance_trend.png` - 距离趋势
- `pace_trend.png` - 配速趋势
- `heart_rate_trend.png` - 心率趋势
- `weight_trend.png` - 体重趋势
- `monthly_summary.png` - 月度统计

填写月度总结：
- 在 `data/2026/01-January.md` 的"本月总结"部分
- 记录完成情况、收获、需要改进的地方

## 常见问题

### Q: 忘记记录某次训练怎么办？

A: 可以手动添加记录，只需在月度文件的表格中添加一行即可。

### Q: 如何修改历史记录？

A: 直接编辑月度文件中的对应行，然后重新运行分析脚本。

### Q: 训练计划可以调整吗？

A: 可以！根据实际情况随时调整训练计划，重要的是循序渐进。

### Q: 心率一直超标怎么办？

A: 继续放慢配速，直到心率达标。宁可慢，不要快。有氧基础需要时间建立。

## 小技巧

1. **训练前截图计划**：训练前用手机截图今日计划，训练时参考
2. **设置心率提醒**：在手表/手环上设置心率上限提醒（138 bpm）
3. **记录及时**：训练后立即记录，避免遗忘
4. **定期备份**：每周将数据推送到 Git 仓库备份
5. **打印月计划**：可以打印月度训练计划贴在墙上
